## Requirements - Download

* Hyprland (Arch)
* Waybar
* btop (You can download it using sudo pacman -S btop)
* pavucontrol
* kitty (You can change it if you don't use kitty terminal)
* peaclock (You can download it using yay -S peaclock) 
* swaync (Download it using sudo pacman -S swaync swaync-client)
* brave (or your own preferred browser)
* JetBrainsMono Nerd Font (or your preferred font)
* wofi or rofi (It's my launcher, you can customize it freely)
